# v0.4.0 Guided Author Workflow — Implementation Plan

**Goal:** Add a Guided Author Workflow layer that answers "Where am I? What's next? Is it safe?" over the existing narrative pipeline.

**Architecture:** Compose with `status.gather_status()`, add typed workflow models, deterministic rules, and CLI commands. No new canonical authority, no LLM routing.

**Tech Stack:** Python dataclasses + Pydantic (existing project conventions), argparse subcommands, HandlerResult protocol.
